# Light Wand

A Persistance of Vision project for the University of Maine Electrical and Computer Engineering department.

## Compiling

    git submodule update --init --recursive
    mkdir build
    cd build
	export PICO_SDK_PATH=/path/to/pico/pico-sdk
    cmake ..
    make
